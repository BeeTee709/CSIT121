Project Overview

Quick summary of what this site includes.

The goal of this project is to design and develop a website that showcases my work from the semester in a well designed, accessible manner. Each page uses structural elements (header, main, footer), a shared external stylesheet, and a layout that adapts to both mobile and desktop screens.

Some of the labs have their own style sheets but I tried to adjust them to fit the overall theme of this website.

You can explore individual labs and read my lessons learned reflection about the course and this final project.

  • Browse all labs and quick summaries on the Labs page.
  • See example images and media on the Gallery page.
  • Read my self-reflection on the Lessons Learned page.
