Reflection

In this assignment, I maintained a consistent theme across multiple pages, Worked with HTML tables to produce a decent looking tierlist. I learned a few new things in this assignment. When i was looking up how to adjst the submit button I found out about the transition property in CSS. I added a simple transition to the submit button to make it feel more interactive and polished. I also learned how to use the textarea element to add a multi-line text input to my contact form. I made sure to style it to fit the overall design of my site. I also had to learn how to use the resize property in CSS to prevent users from resizing the textarea and breaking my layout. Making tables is very repetitive and a bit frustrating to get the styling right but I enojyed this assignment about as much as curuba elderflower redbull.

When I started making the tierlist I thought It would be a great idea to have pictures for all the flavors. Before I even finished downloading all the images I realized it was going to be annoying to have so many images in the table. But, being what I thought was at least 1/2 done i decided to finish and make a seperate tier list just for the names.
