Usage Examples

IndexDB can be used to store data locally in JavaScript applications. Google docs is one example of a web application that uses IndexDB to store data locally. The data is stored in the browser which allows for offline editing that is then synced to the cloud when the user is back online.

While other browser storage options can be used for simple data storage, IndexDB is one of the best options for large amounts of data

Another way it can be used is storing large contacts lists or messages histories that need to be accessed quickly and efficiently. Such as in a messaging app like WhatsApp or Facebook messenger.
